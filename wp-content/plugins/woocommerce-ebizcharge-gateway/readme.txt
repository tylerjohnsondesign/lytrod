== Installation ==
NOTE: EbizCharge plugin is an extension of WooCommerce, so the WooCommerce plugin must be installed and activated in your WordPress site for this plugin to work properly.

1. Download and install EbizCharge plugin using the built-in WordPress plugin installer shared by EbizCharge team.
If you download EbizCharge plugin manually, make sure it is uploaded to `/wp-content/plugins/` and activate the plugin through the `Plugins` menu in WordPress. Or follow the steps below:
Plugins > Add new > Upload plugin > Upload woocommerce-ebizcharge-gateway.zip > Install Now > Activate Plugin.
2. Configure as you see fit, Woocommerce / Settings / Payments / EbizCharge.
3. Set “Enabled” to Yes.
4. Title "This controls the title which the user sees during checkout."
5. Description "This controls the description which the user sees during checkout.".
6. Paste your Security Id into the “Security Id” field (This key must be created in your EBizCharge Merchant Console).
7. Enter your User Id in the “User Id” field. This must match the User Id that was entered when the Security Id was created. If you do not have one, leave this field blank.
8. Enter your Pin in “Pin” field. This must match the User Id that was entered when the Security Id was created. If you do not have one, leave this field blank.
9. Sale Method "Select which sale method to use. Authorize Only pre-authorizes the customer’s card for purchase amount only. Authorize & Capture pre-authorizes the customer’s card and collects funds at the same time".
10. Automatically Capture on Completed Order "If you select "Yes", when you change the order status to "Completed", the payment will be automatically captured".
11. Accepted Cards "Visa / Master Card / AMEX / Discover".
12. Billing Information Storage "Allow customer's to save billing information for future use during checkout".
13. Bank Account Payments "This allows customers to view an option to pay with bank account and save bank accounts as payment methods for future use".
14. Subscriptions "This allows customers to subscribe to products for recurring orders".
15. Subscription Frequencies "Select which frequencies to allow for subscription".
16. EBizCharge Hub Syncing "This allows you to upload your customers, invoices and sales orders to the EBizCharge hub so they can appear and be accessed in the EBizCharge Customer Payment Portal, mobile app, your ERP or other connected software. If this is enabled, any customers, products, or order created outside of WooCommerce (like in the Customer Payment Portal, mobile app and ERP) can also be downloaded/imported into WooCommerce by clicking the download buttons".
17. Item Source "Select whether you use WooCommerce as your source for items and you need to uplaod your items to the EBizCharge hub or whether you use an ERP as your item source and need to download items from your ERP and import them into WooCommerce".
18. Upload Options "Upload Customers / Upload Products / Upload Orders / Upload Invoices".
19. Download Options "Download Customers / Download Products / Download Orders".
20. Click on the “Save Changes” button in bottom.
